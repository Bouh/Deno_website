# Deno_website
Website with Deno


# Require
`.env` file, to add on Root.
```JS
# .env
#development/production
ENV=development
PORT=8000
```

#### Imports
- std, Denon, markdown_wasm, opine, dejs, http_errors

# Start
`denon start` or use `start.bat`

# Format
`denon format` or use `scripts/format.bat`
